\section{Benutzeradministration}\label{sect:useradministration}

Durch einen Klick auf das in Abbildung \ref{img:iconUsers} dargestellte Symbol, das sich in der Navigationsleiste oberhalb jeder Seite findet, gelangt man in das \ATFROGSnosp-Modul zur Benutzeradministration. In diesem Modul lassen sich bestehende Benutzer einsehen und löschen. Außerdem kann das Passwort eines Benutzers auf das in der Konfigurationsdatei \texttt{web.xml} definierte Standardpasswort (siehe Abschnitt \ref{sect:install:konfigurationsdateien}) zurückgesetzt werden.

\subsection{Auswahl eines Benutzers}\label{sect:useradministration:auswahlbenutzer}
Über das mit {\small{\texttt{Benutzername:}}} beschriftete Eingabefeld ist eine Suche nach Benutzern aus der Menge der \OPENXCHANGE-Benutzer möglich. Hierzu muss ein Suchmuster in das Eingabefeld eingetragen werden. Das Sternchen (\texttt{*}) kann als Platzhalter für beliebig lange Zeichenfolgen genutzt werden und darf innerhalb des Suchmusters beliebig oft auftreten.\footnote{In Abbildung \ref{img:user_overview} erscheint als Ergebnis der Suchanfrage \texttt{*ueller*} der Name Hans Mueller -- in der Annahme, dass dessen Benutzername \textit{Mueller} ist. Die Anfragen \texttt{Mueller} und \texttt{*ueller} würden zu demselben Ergebnis führen. Würde ein Benutzer mit dem Benutzernamen \textit{Muellermeyer} existieren, so würde auch dieser in der Ergebnisliste der Anfrage \texttt{*ueller*} erscheinen, in der Ergebnisliste der beiden anderen Anfragen hingegen nicht. Zu beachten ist, dass das Sternchen (\texttt{*}) auch als Platzhalter für eine leere Zeichenfolge stehen kann.} Die Suche wird durch einen Klick auf \includegraphics[scale=0.6]{graphics/button_search2.png} gestartet. Das Suchergebnis wird anschließend in der Liste unterhalb des Eingabefeldes dargestellt (siehe Abbildung \ref{img:user_overview}).\\

   \centering

\subsection{Zurücksetzen des Passworts}\label{sect:useradministration:resetuserpwd}
Vorausgesetzt das Skript \textit{resetuserpwd} ist installiert (siehe Abschnitt \ref{sect:install:editweb.xml}), so kann das Passwort für einen Benutzer zurückgesetzt werden. Dafür muss zunächst einmal der entsprechende Benutzer gesucht und ausgewählt werden (siehe Abschnitt \ref{sect:useradministration:auswahlbenutzer}).\\

In der Detailansicht kann nun durch einen Klick auf \includegraphics[scale=0.6]{graphics/icon_reset.png} das Passwort des zuvor ausgewählten Benutzers zurückgesetzt werden. Nachdem der Wunsch bestätigt wurde, ist die Operation durchgeführt.

\subsection{Löschen eines Benutzers}
Um einen \OPENXCHANGE-Benutzer zu löschen, muss dieser zunächst einmal gesucht und ausgewählt werden (siehe Abschnitt \ref{sect:useradministration:auswahlbenutzer}).\\

In der Detailansicht kann der Benutzer durch einen Klick auf \ICON{graphics/icon_trash.png} gelöscht werden. Nachdem der Wunsch bestätigt wurde, ist der Benutzer endgültig gelöscht.\
